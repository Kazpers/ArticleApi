Article API

A test API for article management

GET
Get All Articles

/api/articles

This gets a list of all articles in the system

Example

curl --location --request GET 'http://localhost:3000/api/articles

GET
Get Article By Id

/api/article/{articleId}

This gets an article by specific id

Example

curl --location --request GET 'http://localhost:3000/api/article/c4c2177b-8465-40dc-a8b8-a706f7d99f35'

GET
Get Articles by Author Id

/api/articles/by/author/{authorId}

This gets a list of all articles by a given author

Example

curl --location --request GET 'http://localhost:3000/api/articles/by/author/ca93fc3b-195c-4293-ac5b-8b49e2c1684c'

GET
Get All Authors

/api/authors

This gets all authors in the system

Example

curl --location --request GET 'http://localhost:3000/api/authors'

POST
Add An Author

/api/author

This will add an author to the system

Headers

Content-Type: application/json

Body

{
    "name": "K. F. Bjerg",
    "phone": "+4555555555",
    "email": "kfb@supermagasinet.dk"
}

Example

curl --location --request POST 'http://localhost:3000/api/author' \
--header 'Content-Type: application/json' \
--data-raw '{
        "name": "K. F. Bjerg",
        "phone": "+4555555555",
        "email": "kfb@supermagasinet.dk"
    }'

POST
Add An Article

/api/article/by/author/{authorId}

This will add an article to the system

Headers

Content-Type: application/json

Body

{
    "topic": "Hvor er mine knapper?!",
    "articleBody": "Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium,
	totam rem aperiam, eaque ipsa quae ab illo inventore veritatis et quasi architecto beatae vitae dicta sunt explicabo.",
    "year": 2020
}

Example

curl --location --request POST 'http://localhost:3000/api/article/by/author/ca93fc3b-195c-4293-ac5b-8b49e2c1684c' \
--header 'Content-Type: application/json' \
--data-raw '{
    "topic": "Hvor er mine knapper?!",
    "articleBody": "Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium,
	totam rem aperiam, eaque ipsa quae ab illo inventore veritatis et quasi architecto beatae vitae dicta sunt explicabo.",
    "year": 2020
}'

DELETE
Delete Article

/api/article/{articleId}

Deletes an article from the system

Example

curl --location --request DELETE 'http://localhost:3000/api/article/c4c2177b-8465-40dc-a8b8-a706f7d99f35'
